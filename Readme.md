# Goal tracker MERN stack

simple goal tracker with CRUD functionality

### tech stack

- reactJS - frontend
- expressJS - backend/server
- mongoDB - database
- redux-toolkit - state management

### features

- user registration
- user login(local storage persist)
- user logout
-  add goals
- delete goals
